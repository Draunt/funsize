data/mozilla-root.crt is used to talk to Balrog server, which uses a
certificated signed by Mozilla CA. See
https://wiki.mozilla.org/MozillaRootCertificate for more details.

data/platforms.js is generated by running data/generate_update_platforms.sh in
the data directory.


data/docker-worker-pub.pem is downloaded from
http://references.taskcluster.net/docker-worker/v1/docker-worker-pub.pem as
stated at
http://docs.taskcluster.net/docker-worker/#encrypted-environment-variables
